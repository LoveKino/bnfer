## 安装

`npm i bnfer --save` 或者 `npm i bnfer --save-dev`

全局安装, 使用 `npm i bnfer -g`



## 使用方法








### API 快速运行



```js
let bnfer = require('bnfer')
let {parse} = bnfer;
let grammer = parse('EXP := num | EXP + num | EXP - num');
console.log(JSON.stringify(grammer, null, 4));
```

```
输出

    {
        "startSymbol": "EXP",
        "T": [
            "num",
            "+",
            "-"
        ],
        "N": [
            "EXP"
        ],
        "productions": [
            [
                "EXP",
                [
                    "num"
                ]
            ],
            [
                "EXP",
                [
                    "EXP",
                    "+",
                    "num"
                ]
            ],
            [
                "EXP",
                [
                    "EXP",
                    "-",
                    "num"
                ]
            ]
        ]
    }

```
